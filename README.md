# semana-tec-oct-2025-Proyecto-juego
Clase el arte de la programación, juego pedido por el profesor funcionando
Juego Pac-Man

## Descripción
Este proyecto corresponde a la actividad práctica de la Semana Tec.
Se utilizó el código base del juego Pac-Man (de Grant Jenks - Free Python Games)
y se realizaron las siguientes mejoras:

1. **Cambio del tablero**: se rediseñó la disposición del laberinto.
2. **Cambio de color**: los puntos son ahora de diferente color al igual que pacman y los fantasmas
3. **Aumento de la velocidad de los fantasmas**: se incrementó para mayor dificultad.

## Validación de código
Se utilizó la herramienta `flake8` para revisar el codigo conforme con PEP8, a pesar de no poder resolverlo.
